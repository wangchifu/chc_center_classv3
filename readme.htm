首頁 / 說明

特別申明：

本系統無償提供彰化縣各編班中心使用，我們沒有受縣府妥託此業務，也不負編班結果之責任，請了解本程式規則，詳加測試後，自行決定採用與否。

功能說明與限制：

1.本程式盡量平均分配「總人數」、「男女生人數」及「所標註的特殊生」到各班，先編班，再編導師。
2.特殊生(1)、雙胞胎同班(2)、雙胞胎不同班(3)等情況，請於 cloudschool 上標註後，方才匯出 xlsx 檔。
3.所匯入的 xlsx 檔，請由 cloudschool 系統產出，並注意名單及人數是否為正確。
4.雙胞胎對應的另一位新生，請注意流水號是否完整正確，如為 A0001，而非 0001。
5.分配到特殊生的班級，系統預設該班酌減一人，請注意是否有班級因此而被分配到一班超過法定上限人數 28 人，尤其是滿編或總量管制的學校又有特殊生時，若超過28人將彈出警告視窗。
6.有上述問題，或是其他特別原因(多胞胎)，可利用動作4，將某班某生移動到指定班級。
7.三胞胎問題，於2021年更新程式已解決，請參考「「三胞胎說明」，因我們無法修改 cloudschool ，故只能修改下載的 xlsx 來解決。
8.老師不教自己小孩，於2022年更新程式已解決，請參考「導師小孩不能同班的設定」
9.雙胞胎、三胞胎又其中成員有特殊生時，無解。
10.特教生綁導師，無解。
11.特教生減二人以上，無解。

程式沿革

1.本程式由二林國小紀老師十多年前首創，免費給予各編班中心使用，須安裝於 sfs3 中使用。
2.2017年，國中新生編班採雙亂數編班，請求學籍小組幫助，由小組獨立此程式出來，服務國中編班。
3.2019年，由和東國小王老師更改為國小獨立版，爾後再修正部分功能。
4.2021年，增加編「三胞胎」功能，詳看「三胞胎說明」。
5.2022年，修改新生上限為28人，增加「導師小孩不能同班的設定」。
